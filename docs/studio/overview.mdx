# Studio Overview — Your Creative Toolkit

**Frnd Studio** is the creative production hub inside frndOS. It's where you turn ideas into final visual assets — with AI handling the heavy lifting while you stay in creative control.

## What's in Studio

Studio is a collection of creative tools, all aware of your Brand Knowledge:

| Tool             | What It Does                                | Best For                            |
| ---------------- | ------------------------------------------- | ----------------------------------- |
| **KV Generator** | Generate Key Visuals from concept to final  | Campaign hero visuals, social posts |
| **Image Editor** | Refine and edit generated visuals           | Detail adjustments, manual touches  |
| **Resizer**      | Adapt one visual to multiple platform sizes | Cross-platform delivery             |

> More Studio tools are rolling out — including Storyboard, Editorial Planner, and Campaign Creator. Stay tuned via the [What's New](/blog) section.

## Where to Find Studio

1. Log in to **frndOS**
2. Select the brand you're working on from the **brand switcher** in the sidebar
3. Open **Frnd Studio** from the sidebar
4. Pick the tool you need

> **Why select the brand first?** Studio tools are brand-aware — they pull from whichever brand is active in the sidebar. Picking the right brand before you start ensures your output inherits the correct Brand Knowledge.

## The Studio Workflow

Most creative work in Studio follows a consistent flow:

```
1. Brand Knowledge (set once per brand)
        ↓
2. Create a Project (campaign-level context)
        ↓
3. Generate in KV Generator (Sketch → Craft)
        ↓
4. Refine in Image Editor (optional)
        ↓
5. Resize for platforms (Resizer)
        ↓
6. Final assets ready
```

## Why Studio is Different from Standalone AI Tools

Most standalone AI image tools start from a blank prompt every time. Studio is different:

* **Brand-aware by default** — every generation inherits your Brand Knowledge
* **Project-scoped** — campaign context (brief, audience, direction) is baked in
* **End-to-end workflow** — concept → refine → resize, all in one platform

This means less prompt engineering, more consistent output, and faster turnaround.

## Who Should Use Studio

Studio is built for the full creative team:

* **Designers & Art Directors** — fast ideation, finalization, and platform adaptation
* **Account Managers & PMs** — explore directions without waiting for design resources
* **Creative Directors** — review concepts and provide direction in real-time
