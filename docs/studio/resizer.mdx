# Resizing Visuals for Different Platforms

The **Resizer** lets you take a single approved Key Visual and adapt it across multiple platform sizes and aspect ratios — without redoing the work from scratch.

## When to Use the Resizer

Use the Resizer when:
- Your KV is **approved** and ready for delivery
- You need to deploy across **multiple platforms** (Instagram, TikTok, banners, etc.)
- You want consistent visual identity across all sizes

Don't use the Resizer for major concept changes — those belong in the [KV Generator](./kv-generator) refine step.

## Before You Start

You need:
- A finalized KV from the KV Generator (in Craft mode)
- Knowledge of which platforms/sizes you're targeting

## How Resize Mode Works

Resize Mode takes your source KV and produces variants at different dimensions:

1. **Open your finalized KV** in the KV Generator
2. Switch to **Resize Mode**
3. Select the platform sizes or aspect ratios you need
4. Generate the resized variants
5. Download or send to the next step in your workflow

## Common Use Cases

| Source KV | Resize Targets |
|---|---|
| Landscape hero visual | Instagram feed (1:1), Stories (9:16), banner ads (various) |
| Square social post | Story format (9:16), portrait feed (4:5) |
| Portrait launch visual | Landscape thumbnail, square feed post |

## FAQ

**Q: Does Resize Mode just crop, or does it intelligently adapt?**
Resize Mode adapts the visual to the target dimensions — not a simple crop. Important elements stay visible and the composition remains balanced.

**Q: Can I resize an externally-made image?**
The Resizer is built into the KV Generator workflow, so the source visual should be a KV you've created in frndOS.

**Q: Can I customize the resize per platform?**
Resize behavior depends on the source KV and target dimensions. For platform-specific tweaks, you can return to Craft mode for that variant.

**Q: What platforms are supported?**
Common social and ad platform dimensions are pre-configured. Check inside Resize Mode for the current list.
