# Resizing Visuals for Different Platforms

The **Resizer** lets you take a single visual — generated in KV Generator or uploaded directly — and adapt it across multiple platform sizes and aspect ratios, without redoing the work from scratch.

## When to Use the Resizer

Use the Resizer when:
- Your KV (or any image you have) is **approved** and ready for delivery
- You need to deploy across **multiple platforms** (Instagram, TikTok, banners, etc.)
- You want consistent visual identity across all sizes

Don't use the Resizer for major concept changes — those belong in the [KV Generator](./kv-generator) refine step.

## Before You Start

You need:
- A finalized KV from the KV Generator (in Craft mode) **or** your own image uploaded into Craft mode — see [Creating Your First Key Visual](./kv-generator)
- Knowledge of which platforms/sizes you're targeting

## How Resize Mode Works

Resize Mode takes your source image and produces variants at different dimensions:

1. **Open your finalized KV — or your uploaded image — in Craft mode**
2. Switch to **Resize Mode**
3. Select the platform sizes or aspect ratios you need
4. Optionally, add a **specific instruction** for how the resize should be handled (see below)
5. Generate the resized variants
6. Download or send to the next step in your workflow

### Adding Custom Resize Instructions

When you resize, you can give the AI **specific instructions** for that resize — not just the target dimensions.

For example, when a resized variant needs adjusting, you can add a prompt like *"keep the logo in the top-right and preserve all text components"* and the AI will regenerate the variant from the base image with that guidance applied to the chosen aspect ratio.

### Interactive Chat in Resize

Resize now supports **interactive chat**. Previously, chat was only active in the Sketch and Craft stages — now you can also chat during resize. When you want to retry or refine a resize, add a follow-up prompt and the AI regenerates from the base image at your target ratio.

> **Note:** Resize uses a fixed engine — you don't choose a model for resizing. Just describe what you want and let it handle the adaptation.

## Common Use Cases

| Source Image | Resize Targets |
|---|---|
| Landscape hero visual (generated or uploaded) | Instagram feed (1:1), Stories (9:16), banner ads (various) |
| Square social post | Story format (9:16), portrait feed (4:5) |
| Portrait launch visual | Landscape thumbnail, square feed post |

## FAQ

**Q: Does Resize Mode just crop, or does it intelligently adapt?**
Resize Mode adapts the visual to the target dimensions — not a simple crop. Important elements stay visible and the composition remains balanced.

**Q: Can I resize an externally-made image?**
Yes. In Craft mode, use **Upload Image** to bring in any image you already have — no generation required. Once uploaded, it behaves like a generated KV, so you can send it straight to Resize Mode.

**Q: Can I customize the resize per platform?**
Yes. You can add specific resize instructions per variant and use interactive chat to refine. For deeper edits, you can also return to Craft mode for that variant.

**Q: Can I set a fully custom size?**
Custom sizes are being rolled out. Note that the underlying model has its own aspect-ratio and resolution limits (a minimum and maximum), so very large or unusual custom sizes may be handled by generating at a supported size and then upscaling. If a custom dimension isn't directly supported, frndOS works toward the closest valid output.

**Q: What platforms are supported?**
Common social and ad platform dimensions are pre-configured. Check inside Resize Mode for the current list. Print-specific specifications (e.g., CMYK, print-ready output) are not part of the current social-media-focused resize flow.
