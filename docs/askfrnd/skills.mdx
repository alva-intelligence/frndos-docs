# Using AskFrnd Skills

**Skills** let you extend what AskFrnd can do. A skill is a reusable **instruction pack** — a set of directions AskFrnd loads when you use it, so it behaves the way you want for a specific task (a brand tone of voice, a repeatable analysis, a writing format).

You can **install** skills others have shared, or **create** your own — from scratch or by importing a public GitHub repo.

> **Availability:** Skills is a beta feature, rolled out gradually. If you don't see the **Skills** tab in Chat Settings, it isn't enabled for you yet.

## Where to Find Skills

Skills live in **AskFrnd → Chat Settings → Skills**:

1. Open **AskFrnd** (from the sidebar or the floating button)
2. Open **Chat Settings**
3. Go to the **Skills** tab

On the Skills tab you'll see your installed skills, each with an on/off toggle, plus buttons to **Create skill** and **Browse directory**. If you have none yet, it reads: _"No skills yet. Create one or browse the directory."_

## Turning Skills On and Off

Each installed skill has a **toggle**. Turn it **on** to let AskFrnd use it, **off** to set it aside without uninstalling. Your changes are applied when you save.

## Installing a Skill from the Directory

The **Skills Directory** is a shared catalog you can install from.

1. On the Skills tab, click **Browse directory**
2. **Search** by name or author, or **filter by tags**
3. Each skill card shows its description and how many times it's been installed (e.g. _"128 installs"_)
4. Click **Install** — once installed, the button shows **Installed**

## Creating Your Own Skill

Click **Create skill**. You have two ways to build one:

### Option A — Manual

Fill in the skill form:

- **Name** — e.g. _"Brand tone of voice"_
- **Description** — a short summary of what the skill does
- **Instructions** — the instruction pack AskFrnd loads when the skill is used
- **Tags** — type a tag and press Enter (helps others find it in the directory)
- **Visibility** — **Private** (_"Only you, plus people and workspaces you share with."_) or **Public** (_"Anyone can find and install this skill."_)

Then click **Create skill** to save.

### Option B — From GitHub

You can import a skill straight from a **public GitHub repository**.

1. In Create, switch to **From GitHub**
2. Paste a **public GitHub URL** — the repo root (`…/owner/repo`) or a path to a skill folder (`…/tree/main/skills/ad-creative`)
3. Import

> **Heads-up:** The repository must be **public**. Private repos can't be imported.

## Managing Your Skills

From **My Skills**, on each skill you own you can:

- **Share skill** — make it available to others
- **Edit skill** — change its name, description, instructions, tags, or visibility
- **Delete skill** — remove it

## FAQ

**Q: I don't see the Skills tab.**
Skills is a beta feature rolled out gradually. If it isn't in Chat Settings yet, it isn't enabled for you — check back later.

**Q: What's the difference between installing and enabling a skill?**
**Installing** adds a skill to your list (from the directory or by creating it). **Enabling** (the toggle) controls whether AskFrnd actually uses an installed skill.

**Q: Who can see a skill I create?**
Depends on its **Visibility**. **Private** is limited to you and the people/workspaces you share with; **Public** is discoverable and installable by anyone in the directory.

**Q: Can I import any GitHub repo as a skill?**
Only **public** repositories. Paste the repo root or a path to a specific skill folder.

**Q: I turned a skill off — is it gone?**
No. Turning the toggle off just stops AskFrnd from using it. To remove it entirely, delete it from My Skills.
