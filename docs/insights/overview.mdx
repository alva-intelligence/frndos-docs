# Insights Overview

**Insights** is the performance tracking pillar of frndOS. It brings together your brand's marketing performance data from multiple channels — Paid Media, Owned Media, Earned Media, and ATL — into a single dashboard.

Instead of jumping between platform-specific dashboards (Meta, TikTok, spreadsheets), Insights consolidates everything so you can see how your brand is performing across the funnel.

## How Insights is Organized

Insights data is structured around **four media types**, each as a separate tab:

| Media Type | What It Tracks | Data Source |
|---|---|---|
| **Paid Media** | Ad performance across digital platforms | Direct connection (Meta, TikTok Ads) |
| **Owned Media** | Organic social media performance | Direct connection (Facebook, Instagram) |
| **Earned Media** | Influencer/PR coverage (non-owned mentions) | Google Sheets |
| **ATL** | Above-the-line media (TV, print, OOH) | Google Sheets |

Plus two cross-cutting views:

- **Overview** — high-level summary across all media types
- **Campaign** — filtered reporting for specific campaigns

## Per-Brand Data

Insights is **scoped per brand**. If you handle multiple brands:

1. Create each brand separately in frndOS
2. Connect data sources independently for each brand
3. Switch between brands using the **brand switcher** in the sidebar

👉 See: [Managing Multiple Brands](../brand-setup/multiple-brands)

## The Funnel View

Across the Paid Media dashboard (and increasingly other tabs), you can filter performance data by **funnel stage**:

- **Awareness**
- **Consideration**
- **Conversion**
- **EV** (Engagement Value, where applicable)

This helps you see how each channel contributes at different stages of the customer journey.

## What You'll See When You Start

If you haven't connected any data sources yet, your dashboards will show **zero values**. This is expected.

Once you connect **at least one data source** (e.g., one Paid Media platform or one Owned Media account), the Overview and relevant tabs will start populating.

## Data Sync Behavior

- **Daily automatic sync** runs from **7 PM through early morning**
- Sync includes deep data (e.g., comments on posts, not just top-line metrics)
- **Manual force-sync** is available if you don't want to wait
- Initial sync may take time depending on data volume (especially Facebook, where comments-of-comments are pulled)
- A sync status indicator is visible per connection; check "last updated" timestamp to confirm freshness

> **Note:** Sync completion notifications (to Lark or push) are not yet implemented. Check sync status manually for now.

## FAQ

**Q: Can I connect multiple ad accounts under one brand?**
Yes. Each brand has one **Core connection** per platform for its primary account. To add additional accounts of the same type — or to reuse data from other brands in your workspace — use **Extended connections**. See [Core vs Extended Connections](./core-extended-connections).

**Q: Why is my Overview tab showing zero?**
You need to connect at least one data source. Once a connection succeeds, the Overview will populate.

**Q: Is X (formerly Twitter) supported?**
Not yet. X support requires a custom developer submission and is on the roadmap.

**Q: Can I export data from Insights?**
Excel export is supported. For deeper analysis, you can also query the data through AskFrnd (chat interface).
